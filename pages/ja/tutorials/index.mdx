# NERO Chainクックブック

NERO Chainクックブックへようこそ – NERO Chain上での開発に役立つ実用的ですぐに使えるレシピ集です。各レシピは、コピー＆ペースト可能なコードとステップバイステップの手順で特定の開発課題を解決するように設計されています。

## 学習パス

チュートリアルは、段階的にスキルを構築できるよう論理的に整理されています。以下の学習パスに従うことをお勧めします：

1. **コントラクトのデプロイから始める** - NERO Chain上でのスマートコントラクトのデプロイの基本を理解する
2. **AAウォレット統合について学ぶ** - アカウントアブストラクションの基盤を設定する
3. **最初のUserOperationsを送信する** - AAウォレットを通じてブロックチェーンと対話し、スポンサード型トランザクションを使用する
4. **支払い方法を探る** - ERC20トークンを使用した代替ガス支払いオプションを実装する
5. **サポートされているトークンを確認する** - ガス支払い用のERC20トークンを扱う
6. **最初の完全なdAppを構築する** - すべての概念を一貫性のあるアプリケーションにまとめる
