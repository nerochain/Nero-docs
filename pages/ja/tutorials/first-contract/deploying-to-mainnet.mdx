## メインネットへのデプロイ手順

### デプロイ前の準備

1. **コントラクトの最終確認**
   - すべてのテストが通過していることを確認
   - セキュリティ監査を実施（推奨）
   - ガス最適化を確認

2. **メインネット用の資金準備**
   - デプロイに必要なNEROトークンを準備
   - ガス代を見積もり

### Hardhatを使用したメインネットデプロイ

1. **環境変数の設定**
```bash
export PRIVATE_KEY=your_private_key
export NERO_MAINNET_RPC=https://mainnet-rpc.nerochain.io
```

2. **Hardhat設定ファイルの更新**
```javascript
// hardhat.config.js
require("@nomicfoundation/hardhat-toolbox");

/** @type import('hardhat/config').HardhatUserConfig */
module.exports = {
  solidity: "0.8.19",
  networks: {
    neroMainnet: {
      url: process.env.NERO_MAINNET_RPC || "https://mainnet-rpc.nerochain.io",
      accounts: process.env.PRIVATE_KEY ? [process.env.PRIVATE_KEY] : []
    }
  }
};
```

3. **デプロイスクリプトの実行**
```bash
npx hardhat run scripts/deploy.js --network neroMainnet
```

### Foundryを使用したメインネットデプロイ

1. **環境変数の設定**
```bash
export PRIVATE_KEY=your_private_key
```

2. **Foundry設定ファイルの更新**
```toml
# foundry.toml
[profile.default]
src = "src"
out = "out"
libs = ["lib"]

[rpc_endpoints]
nero_mainnet = "https://mainnet-rpc.nerochain.io"
```

3. **デプロイスクリプトの実行**
```bash
forge script script/Deploy.s.sol --rpc-url nero_mainnet --broadcast --verify
```

### デプロイ後の確認

1. **コントラクトの検証**
   - NEROスキャンでコントラクトコードを検証
   - 基本的な機能をテスト

2. **モニタリングの設定**
   - コントラクトのイベントとトランザクションを監視
   - アラートシステムの設定（オプション）
